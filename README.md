# Health_System

# 1. 项目概述

## 1.1 项目介绍

随着国家对医疗健康政策的调整，越来越多的新兴健康管理机构涌现，大批的创业者和传统医疗公司纷纷加入，健康管理机构的业务不能快速的成型，投入产出不成比例等问题逐渐浮现。而健康管理系统应运而生，能够充分解决这些问题。
